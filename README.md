## Requisiti
- Linux (testato su Linux Mint che openSUSE)
- Ghostscript (`gs`)
- `notify-send` (parte di `libnotify-bin`)
- `poppler-utils` (per `pdf2ps` e `ps2pdf`)

- Vanno installati i driver del vostro scanner o Stampante multifunzione
